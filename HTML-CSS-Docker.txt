Step 1: Create a project folder
mkdir simple-html-docker
cd simple-html-docker

Step 2: Create the HTML file

Create a file named index.html:

<!DOCTYPE html>
<html>
<head>
    <title>Docker HTML Demo</title>
</head>
<body>
    <h1>Hello! This HTML page is running inside Docker.</h1>
</body>
</html>

Step 3: Create the Dockerfile

Create a file named Dockerfile in the same folder:

FROM nginx:alpine
COPY index.html /usr/share/nginx/html/index.html
EXPOSE 80


✔ This uses nginx to serve your static site.

Step 4: Build the Docker Image

Run this inside the project folder:

docker build -t simple-html .


✔ simple-html is the image name.

Step 5: Run the Docker Container
docker run -d -p 8080:80 --name simple-html-container simple-html

Container port 80 → Host port 8080

Step 6: Open in Browser

Open:

http://localhost:8080
